# SEVEN — сайт парикмахерской

Одностраничный лендинг.

## Запуск

**Windows:** дважды кликните `start.bat`

**Вручную:**

```bash
npm install
npm start
```

Сайт: **http://localhost:3000**

**Админка:** **http://localhost:3000/admin.html** — название, фон, услуги, мастера, контакты. Пароль в `.env` → `ADMIN_PASSWORD`.

**Telegram-бот:** токен в `.env` → `TELEGRAM_BOT_TOKEN`. На сайте ссылка **@BORRY_BOT**. После запуска сервера напишите боту `/start` в Telegram — заявки клиентов будут приходить вам в личку.

Порт можно сменить в `.env` (см. `.env.example`).

## Редактирование контента

Удобнее всего через **админ-панель** (`admin.html`). Либо вручную в `data/site.json`.

После сохранения в админке обновите сайт (**F5**).

## Структура

```
парехмахер/
├── index.html       # Сайт
├── admin.html       # Админ-панель
├── server.js        # Сервер + API
├── data/site.json   # Данные сайта
├── js/site.js       # Загрузка данных на сайт
├── js/admin.js      # Логика админки
├── images/          # Картинки
└── start.bat
```

## Важно

Сайт работает через **Node.js сервер** (`npm start`). Простое открытие `index.html` двойным кликом **не покажет** услуги и мастеров из `site.json`.
